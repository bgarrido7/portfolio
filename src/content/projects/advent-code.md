## About

Advent of Code is a yearly coding challenge that runs every December, where a new programming puzzle is released each day leading up to Christmas. Each problem is a mix of logic, problem-solving, and creativity, and it’s become a popular way for developers to practice and sharpen their skills.

I originally started doing it just to learn a bit of Python. Nothing serious—just curiosity and a way to get more comfortable with coding through small daily problems.

But over time, it slowly turned into a tradition.

Now, every year when Advent of Code comes around, I try to complete it again—but this time in JavaScript. It’s less about finishing everything perfectly and more about the process: showing up each day, solving problems, and seeing how I’ve improved since the last time.

What started as a learning exercise has become something I genuinely look forward to every December.
